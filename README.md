# Retail Inventory Analytics

This project analyzes retail inventory and sales data to assess inventory health,
identify stockout risk, and support operational decision-making.

Tools used:
- SQL (data modeling and data quality checks)
- Python (analysis and feature engineering)
- Power BI (visualization and reporting)
    # Data Model & Measures
      -Source: SQL Server clean view
      -Date dimension created in Power BI
      -Measures defined for revenue, volume, margin, transactions
